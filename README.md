# Credit Card Fraud Detection

## Problem
Credit card fraud causes major financial losses. This project builds a machine learning model to automatically detect fraudulent transactions.

## Dataset
Kaggle Credit Card Fraud Detection dataset.

## Approach
- Data preprocessing and imbalance analysis
- Feature engineering
- Logistic Regression, Random Forest, Neural Network
- Error analysis and explainability

## Outcome
Improved fraud detection with better recall and model understanding.
